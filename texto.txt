Minicurso: Introdução ao Git e GitHub CLI
INFOSACC - 26/11/2025
O que é Git?
Git é um sistema de controle de versão distribuído que ajuda a rastrear mudanças no código fonte durante o desenvolvimento de software.
O que é GitHub CLI?
GitHub CLI ( gh ) é uma ferramenta de linha de comando que traz funcionalidades do GitHub para o seu terminal.
